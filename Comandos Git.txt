git clone linkdorepositorio = clonar o repositorio da net ou de outro computador.
git init = iniciar um novo repositorio.
git add = adicionar arquivos no versionamento do git.
git status= verfificar status atual do repositorio.
git add .= adicionar todos os arquivos da pasta de uma só vez.
git commit -m "comentario"= registrar o trabalho.
git commit -am "comentar"= adiciona e registra o trabalho ao mesmo tempo.
git push remote branch = comando para enviar o codigo para o repositorio remoto."Git Hub".Origin = remote padrao do git.Branch padrao se chama master.
git push origin master= comando padrao para enviar os arquivos para o repositorio.
git config --global= alteração de usuario ou email
git pull origin master =:baixas alterações do repositorio
